## About Me
I'm a self-hosting enthusiast running Proxmox with more than a few open source applications occupying VMs and LXCs. My tech journey started long, long ago in what seems like a galaxy far, far away - spending my early-mid childhood days with my Dad over my shoulder as we learned Atari BASIC language through programming tutorials in Compute! & BYTE Magazine together. Now, after crossing the line into middle age, I spend my free time building real projects to learn modern web and application development, along with exploring containerization and automation. I'm about to venture into the world of modern development with ReactJS, Tailwind CSS, Python etc. (wish me luck!)
